# UrbanSneaks - API REST

API REST desarrollada en Java con Spring Boot para la gestión interna
de una tienda de calzado y ropa urbana.

## Funcionalidades
- Gestión de productos
- Categorías
- Stock
- Ventas
- Usuarios

## Tecnologías
- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## Estructura del proyecto
El proyecto sigue una arquitectura en capas:
- controller
- service
- repository
- dto
- mapper
- exeption
- entity
